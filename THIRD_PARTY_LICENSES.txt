OrchSynth Third-Party Notices
=============================

This file covers third-party components distributed inside the OrchSynth VST3
bundle. It is copied into OrchSynth.vst3 so users can inspect the notices from
the installed plug-in folder.

Steinberg VST3 SDK
------------------

OrchSynth is built as a VST3 plug-in using Steinberg's VST3 SDK. The SDK is
fetched by the build system and is subject to Steinberg's VST3 SDK license
terms. See the VST3 SDK license materials included with the fetched SDK source
